Deque class


This Class Implements the functionality of a Deque/Double Ended Queue using array  of size 10. This class enables the user to interact with the deque array and perform  all the functionalities of Deque. User can Enqeue Elements at the Front  and Rear.  Dequeue elements from front and rear.
